================================================================================

Similarity Threshold:
  - 0.50: Very permissive (not recommended)
  - 0.65: Good for testing
  - 0.75: Recommended for production ← USE THIS
  - 0.85: Very strict
  - 0.95: Extremely strict

Download Settings:
  - IPFS Gateway: https://cloudflare-ipfs.com
  - Library CID: QmQNREjjXTQBDpd69gFqEreNi1dV91eSGQByqi5nXU3rBt
  - Retry attempts: 3
  - Retry delay: 2 seconds

Image Format:
  - Format: NV21 (YUV420SP)
  - Size: (width × height × 3) / 2 bytes
  - Rotation: 0°, 90°, 180°, 270°

Library Files:
  - libmegface.so (Core engine)
  - libFaceDetectCA.so (Detection)
  - libMegviiUnlock.so (Unlock)
  - libMegviiUnlock-jni-1.2.so (JNI)
